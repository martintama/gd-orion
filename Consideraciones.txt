Estrategia:

Logueo.

Se ingresan el usuario y password. Se verifica si todos los datos están completos y se devueve un return_status el que puede ser:

0 - Todo OK: Logueo
1 - Usuario o password incorrecto
2 - Falta completar usuario o password
3 - Usuario baneado
